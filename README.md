# <img src="public/favicon.svg" width="32" height="32" alt="ImmoCalc logo"> ImmoCalc

A vibe coded web-app to quickly calculate actual costs for a property in Germany.

[Demo](https://immo-rechner.philippmeissner.dev/)

Use it when you meet with a bank and discuss different rates. It gived you a detailed overview of the costs to be expected.
It also helps comparing different rates with each other.


## Getting started
* `npm install`
* `npm run start`
